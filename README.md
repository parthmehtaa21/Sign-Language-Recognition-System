# Description
* Dataset used - ASL alphabet on Kaggle - https://www.kaggle.com/datasets/grassknoted/asl-alphabet
* Number of classes - 27 (A-Z and Space)
* Number of images - 89000
* Model Used - LSTM
* Acuuracy - 95%
